# The Custom _printf Project

This is a collaborative project for Holberton School. We created a simplified version of the printf function in C. 

In this project, we learned how to use variadic functions and handle different format specifiers like characters and strings. We focused on making the code clean and following the Betty style.

## How it works:
- It looks for the '%' symbol in the string.
- If it finds 'c', it prints a character.
- If it finds 's', it prints a string.
- If it finds another '%', it just prints the percentage sign.
